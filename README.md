## Welcome to Waqt
<small><em>pronounced "wuh-kth"</em></small>

Have you ever been faced with the problem of not feeling motivated enough to compile and create all ur calendar events in one spot... 
I did too, so I made an AI-powered calendar (specifically GPT 4o). 

- Basically, it translates natural language, and parses it into compiled events that can be added to your custom calendar, with just a click!
